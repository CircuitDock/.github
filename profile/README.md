## Our Mission

CircuitDock exists to make embedded systems development faster, cleaner, and more enjoyable.

Instead of starting every project from scratch, CircuitDock provides reusable hardware, firmware, software, and development tools that allow engineers to prototype and deploy embedded systems in minutes.

Whether you're building with Arduino, STM32, ESP32, Raspberry Pi, RP2040, or custom hardware, CircuitDock aims to provide a common engineering foundation.

# Projects

## CircuitDock Studio

A cross-platform desktop application for embedded development.

Features planned:

- Device Manager
- Serial Terminal
- Firmware Updater
- GPIO Explorer
- I²C Scanner
- SPI Tools
- Logic Analyzer
- Oscilloscope Integration
- Project Manager
- Board Configuration

## CircuitDock SDK

A collection of reusable embedded libraries.

Supported platforms include:

- STM32
- ESP32
- Raspberry Pi Pico
- Arduino
- AVR
- ARM Cortex-M

Libraries include:

- GPIO
- UART
- SPI
- I²C
- CAN
- USB
- Sensors
- Displays
- Storage
- Networking

# Why CircuitDock?

Engineers spend too much time reinventing the basics.

CircuitDock focuses on reusable engineering tools so you can spend more time building products instead of rebuilding infrastructure.

Our philosophy is simple:

- Clean code
- Clean hardware
- Excellent documentation
- Professional tooling
- Open source
